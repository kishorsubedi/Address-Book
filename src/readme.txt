At first, user is asked to enter C/O for creating or opening a new address book.

C 

New addressbook instantiated. User is prompted to add/edit/delete person. New person objects are appended to the arraylist.
At the end, new file is created and arraylist is transferred to new file.

O

Checked if the file is present or not. If present, new addressbook instantiated. The file is streamed to the arraylist of person objects. 
User is prompted to add/edit/delete person. New person objects are appended to the arraylist.
At the end, original file is removed.
New file is created and arraylist is transferred to new file.
